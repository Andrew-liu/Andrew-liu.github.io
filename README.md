## 📖 关于本博客

**Snow Memory** 是我从 2014 年开始维护的个人博客 —— 一共 **157 篇** 涵盖 C++ / Python / Go 源码阅读、系统设计、读书笔记与生活观察的长文。

博客目前覆盖的主要话题：

- **源码剖析**：Google Protobuf、Tornado、Shadowsocks、Redis 订阅发布……
- **系统与网络**：Linux 内核、UNIX 网络编程、Kqueue、Raft、GFS、MapReduce……
- **语言与工程**：C++ / Python / Go / Lua / Clojure / Scala 的读书笔记与踩坑记录
- **工具与环境**：Mac 折腾记、Vim / Tmux / Shell / Git 小结
- **心路札记**：某乎 / 某度 / 某讯的实习与择业思考，每一年的年终总结
